==========
Supporters
==========

.. contents::

Supporters are users which support a petition.

A `supporter` can be created for an anonymous or an authenticated user.
Supporters for anonymous users are created based on the current session id.


Data
====

.. sourcecode:: json

    {
        "id": 1234,
        "firstname": "",
        "lastname": "",
        "phone": "",
        "userId":  "<ref>",
        "petition":  "<ref>"
    }
