==========
Signatures
==========

.. contents::

Signatures are used to sign a petition for a user.

A `signature` can be created for an anonymous or an authenticated user.
Signatures for anonymous users are created based on the current session id.


Data
====

.. sourcecode:: json

    {
        "id": 1234,
        "firstname": "",
        "lastname": "",
        "phone": "",
        "userId":  "<ref>",
        "petition":  "<ref>"
    }
